# 🐝 Beecrowd - Exercícios de Lógica de Programação Java

![Beecrowd](https://img.shields.io/badge/Beecrowd-Exercícios-orange?style=for-the-badge&logo=codeforces&logoColor=white)

## 📚 Descrição

Este repositório foi criado para armazenar minhas soluções dos exercícios de **Lógica de Programação** da **Beecrowd** utilizando a linguagem **Java**.

O objetivo é praticar a resolução de problemas, desenvolver o raciocínio lógico e aprimorar habilidades de programação por meio dos desafios disponibilizados na plataforma.

## 🎯 Objetivos

- Praticar lógica de programação.
- Aprimorar a resolução de problemas.
- Evoluir na escrita de código limpo e organizado.
- Criar uma base sólida para algoritmos e estruturas de dados.
